Superfície de card navy escura com borda fina e barra de destaque opcional. Nunca clara.

```jsx
<Card accent="blue" interactive>
  <h3>Active Diamond Foam</h3>
  <p>Detergente espuma ativa para frotas e máquinas agrícolas.</p>
</Card>

<AccentBlock accent="teal" title="Tecnologia Touchless Real">
  Preserva o brilho de pinturas modernas, eliminando riscos mecânicos.
</AccentBlock>
```

`Card`: `accent` (blue/gold/teal) desenha a barra lateral; `interactive` adiciona lift no hover; `raised` usa navy-light.
`AccentBlock`: o motivo mais recorrente do PDF — azul=performance, teal=tecnologia, dourado=Provisão.
