Toggle para configurações boolean (ativo no catálogo, destaque na home).

```jsx
<Switch checked={active} onChange={setActive} label="Ativo no catálogo" />
<Switch checked={featured} onChange={setFeatured} size="sm" />
```

Trilho navy, preenche azul Nerta quando ligado. `size`: sm | md. Passe `label` para texto à direita.
